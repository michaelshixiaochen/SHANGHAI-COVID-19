# 上海抗疫

#### 介绍

代码说明

#### QRCODE_identification为实现对抗原检测盒的批量识别，增加将识别数据导出为excel文件功能，增加对代码的注释

#### kindergarten 用于给幼儿园班内小朋友是否完成核酸、抗原检测的接龙进行统计
